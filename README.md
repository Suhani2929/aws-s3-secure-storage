# Secure Cloud Storage using AWS S3

## Project Overview
Created a secure cloud storage system using AWS S3 with proper security configurations.

## What I Implemented
- Created S3 bucket in AWS
- Blocked all public access
- Enabled server-side encryption (SSE-S3)
- Uploaded and managed files securely

## Security Measures
- Prevented unauthorized public access
- Ensured data encryption at rest
- Followed least privilege principle

## Outcome
Successfully built a secure storage system preventing data exposure.
